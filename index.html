01 — sobre mim

Quem sou.

Olá! Sou o Leonardo, também conhecido como Cxi5 — developer front-end com 3 anos de experiência a construir interfaces reais para a web.

O meu foco está na camada visual: HTML, CSS, JavaScript e React são as ferramentas que uso no dia-a-dia para criar interfaces rápidas, acessíveis e visualmente marcantes.

Já construí plataformas sociais para músicos independentes, sistemas de gestão e muito mais. Actualmente aceito projectos freelance — fala comigo se tens uma ideia.
